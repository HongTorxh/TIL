<로컬 저장소 명령어>

git init - 폴더에서 git를 사용한다고 초기화해주는 명령어
git add - 뒤에 파일을 붙이는데 그 파일을 추적하겠다는 명령어
git commit - 파일을 하나로 모아서 세이브 데이터를 만드는 명령어 
git status - 현재 상태를 알려주는 명령어
git branch - 현재의 상태를 기준으로 하나의 분기점을 만드는 명령어
git checkout - 특정 branch로 이동시키는 명령어 
git merge - 서로 다른 분기점을 합치는 명령어

<원격 저장소 명령어>

git clone - 원격 저장소의 내용을 로컬 저장소로 가져오는 명령어
git remote - 로컬 저장소의 내용을 원격 저장소로 연결시키는 명령어
git push - 로컬 저장소의 내용을 원격 저장소에 넣는 명령어
git fetch - 원격 저장소의 내용을 로컬 저장소로 가져오기만 하는 명령어 
git diff - 현재 가져온 데이터와 로컬 저장소의 데이터를 비교하는 명령어
git pull - 원격 저장소의 내용을 로컬 저장소로 가져와 적용시키는 명령어
