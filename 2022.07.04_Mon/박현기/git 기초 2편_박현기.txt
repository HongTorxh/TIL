원격 저장소 명령어
git clone
원경 저장소에 있는 원격 저장소의 내용을 로컬 저장소로 가져오는 것
git remote
로컬 저장소에 있는 내용을 원격 저장소에 연결하는 것
git push
로컬 저장소에 있는 내용을 원격 저장소에 넣는 것
git fetch
원격 저장소에 있는 내용을 로컬 저장소로 가져오는데, 가져오기만 하고 적용x
git diff
fetch에서 가져온 데이터와 로컬 저장소에 있는 내용을 비교하는 것
git pull
원격 저장소에 있는 내용을 로컬 저장소로 가져오는데, 적용 가능

수정을 따로 하면 충돌 생김.
